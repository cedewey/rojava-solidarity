Atom Marquee

Dependencies:
Atom Marquee Feature

Description:
The Atom Marquee provides a Marquee Content node type where users can upload an image, set a title, text and read more link url. The Marquee Content is then displayed block created using Entity Field Query. Marquee Content is ordered by post date showing the most recent based on last updated. A Context (atom_marquee_front) is included to insert the marquee block into the home page.

Caveats:
This module makes an image style, view mode and context assumption
